# Space.AI.NET()

C# 콘솔 Space Invaders 스타일 게임

## 실행 방법

1. .NET SDK가 설치되어 있어야 합니다. (https://dotnet.microsoft.com/download)
2. 터미널에서 프로젝트 폴더로 이동합니다.
   ```zsh
   cd /SpaceAINetApp
   ```
3. 아래 명령어로 바로 실행합니다.
   ```zsh
   dotnet run
   ```

## 게임 조작법
- ← : 왼쪽 이동
- → : 오른쪽 이동
- SPACE : 총알 발사
- S : 스크린샷 저장
- Q : 게임 종료

## 기타
- 실행 시 StartScreen이 표시됩니다.
- 콘솔 창 크기를 충분히 확보하면 더 쾌적하게 플레이할 수 있습니다.
- 스크린샷은 `screenshoots` 폴더에 저장됩니다.
